libBERT
-------

libBERT is a C library for encoding and decoding BERT (Binary ERlang Term)
data. Inspired by the Ruby BERT library [1], libBERT aims to provide the
full BERT format to C programmers.

Dependencies
------------

* CMake >= 2.6.0 [2]

References
----------

[1] A BERT (Binary ERlang Term) serialization library for Ruby.
    http://github.com/mojombo/bert
[2] CMake, the cross-platform, open-source build system.
    http://www.cmake.org/
